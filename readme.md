# AIScope

AIScope (Atom Interactive Scope) は分子動力学シミュレーション、動的モンテカルロシミュレーションなどの粒子シミュレーションの計算結果や、密度汎関数理論計算の計算結果を可視化するソフトウェアです。 主に粒子データの可視化が可能ですが、電子密度など一部の場のデータも可視化可能です。 暫定的な機能ですが3Dプリンタ向けのSTL形式のファイルも出力することができます。
